# PRO Registration Requirements

Field requirements and process notes for registering compositions with major performing rights organizations.

---

## Performing Rights Organizations Overview

A PRO licenses public performances of musical compositions and distributes royalties to songwriters and publishers. Registration is required for royalties to flow. Key distinction: PROs cover the **composition** (words + music) — not the sound recording (that's the label/distributor side).

The major PROs by territory:

| Territory | PRO(s) |
|---|---|
| United States | ASCAP, BMI, SESAC (invite-only), GMR (invite-only) |
| United Kingdom | PRS for Music |
| Canada | SOCAN |
| Australia | APRA AMCOS |
| Germany | GEMA |
| France | SACEM |
| Sweden | STIM |
| Japan | JASRAC |
| International (mechanical) | MLC (US), MCPS (UK), CMRRA (Canada) |

A songwriter or publisher registers with the PRO in their home territory. Reciprocal agreements between PROs ensure royalties flow for uses in foreign territories.

## ASCAP Registration Requirements

**Portal:** ascap.com/music/member (member login required)

**Work registration fields:**
| Field | Required | Notes |
|---|---|---|
| Song Title | Yes | As it appears on the release |
| Alternative Titles | No | Cover titles, foreign titles |
| Composer(s) | Yes | Legal name(s) + IPI number(s) |
| Lyricist(s) | If separate from composer | |
| Publisher(s) | Yes | Legal entity name + IPI number |
| Writer Shares | Yes | Must sum to 100% (writer side) |
| Publisher Shares | Yes | Must sum to 100% (publisher side) |
| ISWC | No | Assigned by ASCAP if not provided |
| ISRC | Recommended | Links recording to composition |
| Genre | Recommended | |
| Language | No | |
| Release Date | Recommended | |

**IPI numbers:** ASCAP assigns IPI numbers to members at registration. Every writer and publisher must have an IPI to receive payments. Find your IPI in the ASCAP member portal under Account Settings.

**Timing:** Register compositions before or on the release date for full royalty coverage from day one.

## BMI Registration Requirements

**Portal:** repertoire.bmi.com (member login required)

**Work registration fields:**
| Field | Required | Notes |
|---|---|---|
| Title | Yes | |
| Alternate Titles | No | |
| Writers | Yes | Full name + BMI CAE/IPI number |
| Publisher | Yes | Publisher name + BMI CAE/IPI number |
| Writer Shares | Yes | Sum to 100% |
| Publisher Shares | Yes | Sum to 100% |
| ISWC | No | BMI assigns on registration |
| ISRC | Recommended | |
| Genre/Music Type | Recommended | |
| Language | No | |

**CAE vs IPI:** BMI historically used "CAE" numbers; these are now unified with IPI. If you have an older BMI CAE number, it's the same as your IPI — same number, different label.

**Co-writes with ASCAP/BMI members:** If a BMI songwriter co-writes with an ASCAP songwriter, each registers the composition with their own PRO. Both societies will pay their respective member based on the agreed percentage split.

## SESAC Registration Requirements

**Portal:** sesac.com (invitation-only membership; contact SESAC directly)

SESAC is selective and invitation-based. If a publisher or songwriter is a SESAC member:
- Registrations are handled through a SESAC member representative
- Requires the same core fields as ASCAP/BMI: title, composers, publishers, shares, ISRC/ISWC links

SESAC's smaller membership means distributions can be higher per-performance for members whose repertoire is heavily used by SESAC licensees (restaurants, venues, broadcasters).

---

## PRS for Music (UK)

**Portal:** prsformusic.com (member login required)

**Additional fields vs. US PROs:**
- `Publisher Admin Territory` — specify if the publisher administers rights only in certain territories
- `Original Language` — required for non-English works
- `Type of Work` — `Musical`, `Serious` (classical), `Dramatic` (musical theatre)

PRS handles both **performance rights** (PRS) and **mechanical rights** (MCPS) under one organization for UK market. Mechanical registration is handled through the MCPS side of PRS.

## Composer Role Codes

Used across most PRO registration systems to define each contributor's role in the composition:

| Code | Role | Description |
|---|---|---|
| C | Composer | Music only (no lyrics) |
| CA | Composer/Author | Music and lyrics |
| A | Author | Lyrics only |
| AR | Arranger | New arrangement of existing composition |
| E | Publisher | Music publisher administering rights |
| SE | Sub-Publisher | Sub-publisher administering in specific territory |
| TR | Translator | Translator of lyrics into another language |
| ES | Entire Series | Publisher acquiring all rights |

## IPI Number Lookup

If IPI numbers are unknown:
- **For ASCAP members:** ascap.com/music/repertory → search by name → IPI displayed in results
- **For BMI members:** repertoire.bmi.com → search by writer name
- **For PRS members:** prsformusic.com/works → writer search
- **International:** CISAC IPI database (ipi.cisac.org) — covers all societies globally

---

## Mechanical Licensing (US)

In the US, mechanical royalties for audio streams and downloads are administered by the **Mechanical Licensing Collective (MLC)** under the Music Modernization Act. Separately from PRO performance royalties:

- Register compositions at themlc.com (free)
- Required fields: song title, ISWC, composer(s), publisher(s), shares
- MLC collects mechanical royalties from DSPs and distributes to publishers/songwriters
- Registration in the MLC and PRO are separate — both are required for full royalty coverage
