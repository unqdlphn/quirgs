# Pre-Release Compliance Checklist

Gate checklist to complete before distributing any release. Organized by category — complete all sections before submitting to a distributor.

---

## Section 1 — Metadata Integrity

| Check | Status |
|---|---|
| Track title is correct and in canonical format | ☐ Pass ☐ Fix |
| ISRC is valid format and confirmed for this recording | ☐ Pass ☐ Fix |
| UPC is valid (check digit verified) | ☐ Pass ☐ Fix |
| Release date is confirmed across all distribution channels | ☐ Pass ☐ Fix |
| Genre is set and matches a platform-standard category | ☐ Pass ☐ Fix |
| Explicit flag is accurately set | ☐ Pass ☐ Fix |
| Duration metadata matches the actual audio file | ☐ Pass ☐ Fix |
| Artist name is in the correct canonical format (no all-caps, no typos) | ☐ Pass ☐ Fix |
| P-line and C-line are complete | ☐ Pass ☐ Fix |
